## 🌟 期末课程设计：教室资源管理系统

位于 `sys/` 目录下。这是一个基于 C++17 实现的高性能教室资源管理系统，采用 **C++ 后端 + MySQL 数据库 + Web 前端** 的架构。

### 1. 系统简介

本项目不依赖庞大的第三方 Web 框架，而是使用 C++ 手工封装了 HTTP 服务器和数据库连接池，实现了从底层网络通信到上层业务逻辑的完整开发。

### 2. 核心功能

* **资源管理**：教室、设备、课程的增删改查及筛选。
* **智能排课**：支持自动冲突检测（时间/教室/教师资源冲突）的排课功能。
* **课表查询**：支持教师课表、学生课表及空闲教室查询。
* **数据可视化**：仪表盘集成 Chart.js，展示教室利用率统计图表。

### 3. 技术栈

* **后端**: C++17 (自定义 HTTP Server, JSON 解析)
* **数据库**: MySQL 8.0
* **前端**: HTML5 / CSS3 / JavaScript / Bootstrap 5
